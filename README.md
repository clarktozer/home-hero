# home-hero

An example home booking app inspired by https://www.tinyhouse.app/

Stack:

-   React.js
-   Redux
-   TypeScript
-   GraphQL
-   Node.js
-   Apollo Client/Server
-   TypeGraphQL
-   TypeORM
-   PostgreSQL
-   Redis
-   Material UI
