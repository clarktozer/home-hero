# home-hero

An example home booking app

Stack:

-   React.js
-   Redux
-   TypeScript
-   GraphQL
-   Node.js
-   Apollo Client/Server
-   TypeGraphQL
-   TypeORM
-   PostgreSQL
-   Redis
