# home-hero

An example home booking app

Stack:

-   React.js
-   Redux
-   TypeScript
-   GraphQL
-   Node.js
-   Apollo Client/Server
-   TypeGraphQL
-   TypeORM
-   PostgreSQL
-   Redis

Integrations:

-   Google auth, maps
-   Facebook auth
-   GitHub auth
-   Stripe
-   Cloudinary

Additional Front-end libraries:

-   Material UI
-   React Router
